# AI Agent Skills

This repository contains a collection of skills and capabilities for AI agents. These skills are designed to be modular and reusable across different AI applications.

## Installation

You can easily install it using the [`skills` CLI](https://github.com/vercel-labs/skills):

```bash
npx skills add fityannugroho/skills
```

Install specific skill:

```bash
npx skills add fityannugroho/skills --skill skill-name
```

Otherwise, just **copy the skill files** from this repository into your project's skills directory.
