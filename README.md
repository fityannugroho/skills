# AI Agent Skills

This repository contains a collection of skills and capabilities for AI agents. These skills are designed to be modular and reusable across different AI applications.

## Installation

You can easily install it using the [`skills` CLI](https://github.com/vercel-labs/skills):

```bash
npx skills add fityannugroho/skills
```

Install specific skill:

```bash
npx skills add fityannugroho/skills --skill skill-name
```

Otherwise, just **copy the skill files** from this repository into your project's skills directory.

## Skills

- [background-exec](background-exec) - Execute  long-running commands in the background using tmux.
- [git-sync](git-sync) - Synchronize local git repository with the remote.
- [node-deps-upgrade](node-deps-upgrade) - Easily upgrade your Node.js project dependencies with `ncu`.
- [toon](toon) - Write data in TOON format, encode JSON to TOON, decode TOON to JSON.

### Another skills

- [csv-diff](https://github.com/fityannugroho/csv-diff) - Compare two CSV files and export the differences to `.diff` file.
- [img-dl](https://github.com/fityannugroho/img-dl) - Download images from URLs.
